# BE-014: Parent Report Email Sending

## Title
Implement automatic email sending for parent reports

## Scope

### In Scope
- Send report email after report creation
- Email includes report link and PIN
- Mark report as sent in database
- Error handling for email failures

### Out of Scope
- Report generation (BE-013)
- Report viewing (BE-013)

## Technical Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Email Service | Resend | Already integrated in BE-005 |
| Trigger | Inline after report creation | Simplest, MVP appropriate |
| Retry | Manual resend only | MVP scope |

## Email Retry Convention

**MVP Approach**: Manual retry only via `/reports/{reportId}/resend` endpoint.

| Scenario | Behavior |
|----------|----------|
| Email fails | Report still created, message indicates "email failed" |
| Parent requests resend | Coach uses "Resend Report" in UI |
| New PIN | Generated on each resend for security |

**Post-MVP Enhancement**: Add automatic retry with exponential backoff (1s, 5s, 30s) using background tasks.

## Acceptance Criteria

- [x] Email sent automatically when report is created
- [x] Email contains report link with report ID
- [x] Email contains 6-digit PIN in clear text
- [x] Email is parent-friendly (not technical)
- [x] Report marked as sent with timestamp
- [x] Email failure doesn't crash report creation
- [x] Coach can resend report email

### services/email.py (additions)
```python
from app.templates.report_email import get_report_email_html, get_report_email_subject

async def send_report_email(
    parent_email: str,
    athlete_name: str,
    coach_name: str,
    report_id: str,
    pin: str
) -> bool:
    """Send parent report email with link and PIN"""
    report_url = f"{settings.frontend_url}/report/{report_id}"

    html = get_report_email_html(
        athlete_name=athlete_name,
        coach_name=coach_name,
        report_url=report_url,
        pin=pin
    )

    subject = get_report_email_subject(athlete_name)

    return await send_email(
        to=parent_email,
        subject=subject,
        html=html
    )
```

### routers/reports.py (modifications)
```python
from app.services.email import send_report_email

@router.post("/{athlete_id}/send", response_model=ReportSendResponse)
async def send_report(
    athlete_id: str,
    user: User = Depends(get_current_user)
):
    """
    Generate report, store with PIN, and send to parent.
    Returns PIN for coach reference.
    """
    athlete_repo = AthleteRepository()
    report_repo = ReportRepository()

    # Validate ownership
    athlete = await athlete_repo.get_if_owned(athlete_id, user.id)
    if not athlete:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail="Athlete not found"
        )

    try:
        content, metadata = await generate_report_content(user.id, athlete_id)
    except ValueError as e:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )

    # Generate PIN and create report
    pin = generate_pin()
    report = await report_repo.create_report(
        coach_id=user.id,
        athlete_id=athlete_id,
        content=content,
        assessment_ids=metadata["assessment_ids"],
        pin=pin,
    )

    # Send email to parent
    email_sent = await send_report_email(
        parent_email=athlete.parent_email,
        athlete_name=athlete.name,
        coach_name=user.name,
        report_id=report.id,
        pin=pin,
    )

    if email_sent:
        await report_repo.mark_sent(report.id)

    return ReportSendResponse(
        id=report.id,
        pin=pin,
        message="Report sent to parent" if email_sent else "Report created but email failed",
    )

@router.post("/{report_id}/resend")
async def resend_report(
    report_id: str,
    user: User = Depends(get_current_user)
):
    """Resend report email (generates new PIN)"""
    report_repo = ReportRepository()
    athlete_repo = AthleteRepository()

    report = await report_repo.get(report_id)
    if not report:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail="Report not found"
        )

    # Validate ownership
    if report.coach_id != user.id:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="Access denied"
        )

    athlete = await athlete_repo.get(report.athlete_id)
    if not athlete:
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail="Athlete not found"
        )

    # Generate new PIN
    new_pin = generate_pin()
    await report_repo.update(report_id, {
        "access_pin_hash": report_repo.hash_pin(new_pin)
    })

    # Send email
    email_sent = await send_report_email(
        parent_email=athlete.parent_email,
        athlete_name=athlete.name,
        coach_name=user.name,
        report_id=report.id,
        pin=new_pin,
    )

    if email_sent:
        await report_repo.mark_sent(report.id)

    return {
        "pin": new_pin,
        "message": "Report resent" if email_sent else "Email failed",
    }
```
